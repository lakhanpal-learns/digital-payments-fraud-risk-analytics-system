# Digital Payments & Fraud Risk Analytics System

## Project Overview

### Introduction

The **Digital Payments & Fraud Risk Analytics System** is an end-to-end fintech analytics project developed to simulate a modern digital payment ecosystem and provide business intelligence for payment operations, fraud monitoring, merchant performance, and customer transaction analysis.

The project combines synthetic transaction generation, relational database management, SQL-based analytics, and interactive Power BI dashboards to replicate the workflow of a fintech data analyst.

---

# Project Objectives

* Build a realistic digital payments dataset using Python.
* Store transaction data in PostgreSQL for structured analysis.
* Perform operational and fraud-related analysis using SQL.
* Develop interactive Power BI dashboards for business decision-making.
* Demonstrate practical fintech analytics skills required for Data Analyst and FinTech Analyst roles.

---

# Business Objectives

The project focuses on answering key business questions, including:

* How healthy is the payment ecosystem?
* What is the transaction success and failure rate?
* Which payment methods generate the highest transaction volume?
* Which cities experience the highest payment activity?
* Where are fraud risks concentrated?
* Which merchants contribute the highest revenue?
* What operational factors affect payment reliability?

# Project Workflow

1. Generate realistic synthetic payment transaction data.
2. Validate and clean the generated dataset.
3. Load transaction data into PostgreSQL.
4. Perform SQL-based operational and fraud analytics.
5. Create business KPIs and DAX measures.
6. Build four interactive Power BI dashboards.
7. Generate business insights and recommendations.

---

# Dashboard Modules

The project consists of four business dashboards:

* Executive Dashboard
* Fraud Risk Analytics Dashboard
* Payment Operations Dashboard
* Merchant & Customer Insights Dashboard

Each dashboard is designed for a different business audience and supports data-driven decision making.
